About Me

Hi, I’m Yuveer Chetraj

I’m a DevSecOps Engineer and Cloud Security Specialist based in Cape Town, South Africa, with 4+ years of hands-on experience securing cloud-native applications across the entire software development lifecycle.

What I Do

I specialize in embedding security into every stage of development, from code to production:

  • Application Security at Scale - Built and scaled comprehensive security programs across 4,000+ repositories, integrating SAST, SCA, secrets scanning, IaC analysis, and container security from IDE to runtime
  • Vulnerability Management - Led enterprise vulnerability programs reducing critical API vulnerabilities to zero through systematic triage, prioritization, and automated remediation tracking
  • CI/CD Pipeline Security - Integrated security controls throughout pipelines using SonarQube, Nexus IQ, Prisma Cloud, and Semgrep OSS, enforcing security gates at each SDLC stage
  • Cloud Security Architecture - Configured CSPM and AISPM across AWS and Azure environments with automated remediation, plus Cloudflare WAF enterprise deployments
  • Runtime Security - Deployed security defenders across AKS/EKS/ECS clusters with MITRE ATT&CK-based threat detection and continuous monitoring
  • Security Culture Building - Delivered Security Champion programs and worked with 30+ development teams to foster security-first engineering practices

My Approach

Security shouldn’t be a bottleneck - it should be an enabler. I believe in:

  • Shift Left Security - Catching vulnerabilities early in the development process through automated scanning in PRs and CI/CD
  • Zero-Touch Automation - Building self-service security tools that eliminate manual bottlenecks while maintaining rigor
  • Developer Empowerment - Partnering with engineering teams through clear guidance, automated workflows, and actionable feedback
  • Continuous Improvement - Using data-driven insights to optimize security processes and reduce risk systematically
